Project 01

Cipher & Signal

A browser-based puzzle-solving suite for CTF players, ARG hunters, and web riddle enthusiasts. Zero tracking — your data never leaves your browser.

◆ Cipher & Signal

Image analysis · Audio forensics · Classical ciphers · Research notebook

Why I Built This

Puzzle communities — from Cicada 3301 to modern ARGs — rely on scattered, janky tools that haven't been updated in years. I wanted a single, modern workspace where you could analyze an image for hidden data, decode a cipher, run frequency analysis on audio, and keep notes on your progress, all without switching tabs or uploading sensitive files to random servers. Privacy-first, completely client-side.

Core Capabilities

◐
Image Suite

Brightness, contrast, RGB channels, steganography, LSB extraction, EXIF metadata, bit-plane analysis

◠
Audio Lab

Waveform & spectrogram visualization, reverse playback, DTMF detection, frequency analysis

◆
Cipher Engine

Caesar, Vigenère, Atbash, Rail Fence, XOR brute-force, Affine, Beaufort, substitution

◇
Encoding Hub

Base64/32, hex, binary, URL, HTML entities, SHA/MD5 hashing, auto-detect decoder

◈
Visual Ciphers

Morse, Braille, Pigpen, Semaphore, NATO phonetic, Tap code, Bacon's cipher

◳
Puzzle Notebook

Track puzzles across platforms, record solutions, markdown notes, JSON export/import

JavaScript CSS HTML Client-Side Only Vercel

Future Roadmap

Collaborative puzzle-solving with shared notebooks and real-time sync
AI-assisted pattern recognition for cipher identification
Browser extension for quick analysis of images/text on any webpage
Community cipher challenge platform with leaderboards

Project 02

Tab Tutor

An AI-powered guitar learning workspace — transcriptions, technique breakdowns, theory context, and structured practice plans for intermediate players.

♬ Tab Tutor

Guitar tab workspace · AI analysis · Practice tracking · Music theory engine

Why I Built This

Learning guitar solos from tabs alone is like reading sheet music without a teacher. You get the notes, but not the why — why this scale, why this technique, why this fingering. Tab Tutor bridges that gap by combining traditional tab notation with AI-powered analysis that breaks down every solo into technique-by-technique lessons, theory explanations, and structured practice plans with tempo milestones. It's the guitar teacher that's always available, always patient, and deeply understands music theory.

Core Capabilities

🎸
Smart Transcription

Drop in Guitar Pro files or tabs — get auto-analyzed breakdowns with technique tags and theory context

📋
Structured Lessons

Self-contained lesson files for each practice session, with tempo milestones and checkpoint assessments

🧠
Theory Engine

Scales, modes, harmonic analysis — understand the "why" behind every note in the solo

📈
Progress Tracking

Practice logs, context files, and tempo progression tracking across all songs

🎵
Song Library

Megadeth, Dream Theater, Guthrie Govan, Gary Moore — prog metal to blues, all analyzed

🔌
Video Learning

Paste a YouTube lesson URL — get a structured learning guide with concept cross-references

Python TypeScript FastAPI AI/LLM Integration Vercel
